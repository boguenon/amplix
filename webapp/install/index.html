Package Installation

1. Pre-requirements for amplix

Install does not meet required environment to proceed.
Please check bellow steps for install and try again.

  • Install the Java SDK(1.8.0 or later).
  • Install MYSQL database
  • Download and extract file.
    Download amplixbi.zip and extract to folder.

    unzip amplixbi.zip

    Copy extracted folder to application installation location, for example /opt
    mv amplixbi /opt/

2. Prepare database

  • Install and Start MySQL database
  • Login as root user with the following command.

    mysql -uroot -p

  • Create empty database with the following command on the above database command shell.

    CREATE DATABASE amplix_meta;
    CREATE USER 'amplix'@'localhost' IDENTIFIED BY 'password';
  • Grant permission to database

    GRANT ALL PRIVILEGES ON amplix_meta.* TO 'amplix'@'localhost';
    FLUSH PRIVILEGES;
  • Create Sample database user and database (Optional)

    CREATE DATABASE foodmeart;
    CREATE USER 'foodmart'@'localhost' IDENTIFIED BY 'password';

    Grant permission to foodmart user and database

    GRANT ALL PRIVILEGES ON foodmart.* TO 'amplix'@'localhost';
    GRANT ALL PRIVILEGES ON foodmart.* TO 'foodmart'@'localhost';
    FLUSH PRIVILEGES;

3. Start Application.

  • Open Shell Command prompt. Execute following command.

    ./bin/RUN_startup.sh
    ./bin/RUN_startup.bat
  • Open this installation wizard on web browser, and follow instruction.

    http://localhost:8080/amplix/install/

4. Stop Application.

  • To stop application, run the following command.

    ./bin/RUN_shutdown.sh
    ./bin/RUN_shutdown.bat

All-in-One Installation

1. Pre-requirements for amplix

Install does not meet required environment to proceed.
Please check bellow steps for install and try again.

  • Install the Java SDK(1.8.0 or later).
  • Install WebApplication Server (Tomcat)
  • Install database for MetaData
  • Download and extract file.
    Download amplixbi.zip and extract to folder.

    unzip amplixbi.zip

    Copy extracted folder to application installation location, for example /opt
    mv amplixbi /opt/

2. Prepare database

  • Install and Start MySQL database
  • Login as root user with the following command.

    mysql -uroot -p

  • Create empty database with the following command on the above database command shell.

    CREATE DATABASE amplix_meta;
    CREATE USER 'amplix'@'localhost' IDENTIFIED BY 'password';
  • Grant permission to database

    GRANT ALL PRIVILEGES ON amplix_meta.* TO 'amplix'@'localhost';
    FLUSH PRIVILEGES;
  • Create Sample database user and database (Optional)

    CREATE DATABASE foodmeart;
    CREATE USER 'foodmart'@'localhost' IDENTIFIED BY 'password';

    Grant permission to foodmart user and database

    GRANT ALL PRIVILEGES ON foodmart.* TO 'amplix'@'localhost';
    GRANT ALL PRIVILEGES ON foodmart.* TO 'foodmart'@'localhost';
    FLUSH PRIVILEGES;

3. Deploy amplix analysis application for your Tomcat Server.

  • Edit web.xml file
    Open the web.xml file in /amplixbi/webapp/WEB-INF/web.xml.
    Edit APP_HOME for the folder that extracted. and uncomment the section

    <context-param>
        <param-name>APP_HOME</param-name>
        <param-value>/opt/amplix</param-value>
    </context-param>
  • Register Application
    On tomcat need to register context.xml on tomcat configuration folder.
    Create below file if not exists on your tomcat folder
    TOMCAT/conf/Catalina/localhost/amplix.xml

    <?xml version="1.0" encoding="UTF-8"?>
    <Context docBase="/opt/amplix/webapp"
             cachingAllowed="true" reloadable="false">
        <Parameter name="APP_HOME" value="/opt/amplix" override="true"/>
    </Context>

4. Deploy and run the web appication

  • Restart Apache Tomcat.
  • Open this installation wizard on web browser, and follow instruction.

    http://localhost:8080/amplix/install/
