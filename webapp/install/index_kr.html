Amplix BI 인스톨 마법사

Amplix BI 인스톨 마법사에 오신것을 환영합니다.

보안상의 이유로 인스톨은 신규 설치 서버에만 적용되게 되어 있습니다. 재설치를 위해서는 패키지 파일을 다시 다운로드 받아 초기상태에서 진행해 주십시오.

인스톨 마법사를 통해 설정과 설치에 대한 단계를 안내하고 있습니다. 단계를 따라 진행해 주십시오.

AmplixBi 를 이용해 주셔서 감사합니다.

성공적인 설치를 위해 다음 필수사항 및 단계를 읽고 진행해 주십시오.

패키지 인스톨
패키지 인스톨

1. 필수 소프트웨어 설치

다음 필수 소프트웨어 설치 없인느 설치를 진행할 수 없습니다.
성공적인 설치를 위해 다음 소프트웨어가 설치되어 있는지 점검하고 다시 시도해 주십시오.

  • Java SDK(1.8.0 또는 이후버전).
  • MYSQL 데이터베이스
  • 패키지 파일 압축 풀기.
    AMPLIX 패키지 파일을 특정 폴더에 압축을 해제합니다.

    unzip amplixbi.zip

    압축 해제한 패키지 파일을 어플리케이션 설치 폴더에 이동합니다. /opt 폴더에 설치하는 경우 다음과 같습니다.
    mv amplixbi /opt/

2. 메타 데이터베이스 준비하기

  • MySQL 데이터베이스를 설치하고 시작해 주십시오.
  • root 사용자로 데이터베이스 관리툴에 로그인하여 다음 명령을 실행해 주십시오.

    mysql -uroot -p

  • 데이터베이스 명령 셀에서 다음 명령을 실행하여 빈 데이터베이스를 생성해 주십시오.

    CREATE DATABASE amplix_meta;
    CREATE USER 'amplix'@'localhost' IDENTIFIED BY 'password';
  • 데이터베이스 접속 권한을 부여해 주십시오.

    GRANT ALL PRIVILEGES ON amplix_meta.* TO 'amplix'@'localhost';
    FLUSH PRIVILEGES;
  • 샘플 데이터베이스와 사용자를 생성해 주십시오.

    CREATE DATABASE foodmeart;
    CREATE USER 'foodmart'@'localhost' IDENTIFIED BY 'password';

    샘플 데이터베이스 사용자 및 데이터베이스 접근 권한을 부여해 주십시오.

    GRANT ALL PRIVILEGES ON foodmart.* TO 'amplix'@'localhost';
    GRANT ALL PRIVILEGES ON foodmart.* TO 'foodmart'@'localhost';
    FLUSH PRIVILEGES;

3. 어플리케이션 시작.

  • 쉘 커맨드 창을 열고 다음 명령을 실행해 주십시오.

    ./bin/RUN_startup.sh
  • 인스톨 마법사를 서버 접속 가능한 브라우져에서 다음 URL 을 통해 접속하고 설치 단계를 따라서 진행하십시오.

    http://localhost:8080/amplix/install/kr

4. 어플리케이션 중지.

  • 어플리케이션을 중지하려면 다음 명령을 쉘 커맨드 창에서 실행해 주십시오.

    ./bin/RUN_shutdown.sh
