시스템 개요

AMPLIX 서버는 웹브라우저를 통한 사용자의 접근 제어, 서비스 처리를 수행하는 역할을 합니다. 이를 위해 내부 데이터베이스 시스템에 SQL 쿼리를 통해 데이터 결과를 수신하고 결과에 대한 포맷, 피벗 변환 등의 작업을 수행한 후 최종 사용자의 화면에 그리드 / 차트의 형태로 데이터를 표현한다.

하드웨어 요구사항

서버 구동을 실행하는 단일 서버 머신 :

- windows 10 server (64비트)

- 2.4GHZ 쿼드 코어 i5 이상 (8세대 인텔 i5 또는 호환되며 동일 성능 이상의 CPU)

- 16GByte RAM

- 20 GB 의 디스크 여유 공간

- 1GB 이더넷 카드 혹은 이상의 네트워크 성능

사용자 PC 시스템 :

- Windows 10 (32bit)

- 구글 크롬 브라우져

- 50 GB 의 디스크 여유공간

- 8GByte RAM

- 모니터 해상도 : 일반 사용자 (1280 x 720 이상), 관리자 및 리포트 개발자 (1920 x 1080 이상)

네트워크 요구사항

AMPLIX 서버는 사용자 PC 의 요청을 HTTP 혹은 HTTPS 프로토콜로 통신을 호출하고 있습니다. 이를 위한 네트워크 포트 및 TCP/IP 연결이 될수 있도록 네트워크 방화벽, 라우팅 접근 가능하도록 설정한 후 진행하여야 한다. 사용하는 포트 번호는 8580 포트를 사용한다.

소프트웨어 요구사항

서버 설치를 위해서 필요한 소프트웨어는 다음과 같다.

- 오라클 자바 JDK 버전 1.8 이상 (1.8.0_261 이상, JDK)

- MySQL 데이터베이스 버전 8 이상 (8.0.21 이상, 네트워트 TCP/IP 포트 3306 이 개방되어 있을 것)

소프트웨어 설치

AMPLIX 서버를 사용하려면 먼저 필요 소프트웨어를 설치 구성하여야 합니다. 소프트웨어 요구사항 섹션에 정의된 필수 소프트웨어가 시스템에 설치되어 있는지, 또한 버전을 확인 한 후 설치를 진행하여야 합니다.

또한 먼저 AMPLIX 소프트웨어를 구매한 후 라이센스 파일을 발급 받아야 사용가능합니다. 라이센스 파일의 신청 및 구매를 위해서는 www.amplixbi.com 을 방문하여 문의할 수 있습니다.
