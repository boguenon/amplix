7.2. 결과 시트 위젯 피벗

결과 시트 위젯 피벗 변경

결과 시트 위젯의 피벗을 수정/변경하려면 위젯 툴바의 피벗 편집 버튼을 클릭하여 피벗 정보 화면을 통해 수정한다.

(그림 : 결과 시트 피벗 편집 화면 레이아웃)

피벗 편집 화면의 각 레이아웃별 설명은 다음과 같다.

-. 비즈니스 오브젝트 테이블 / 인라인뷰 항목 : 선택된 비즈니스 오브젝트의 테이블 / 인라인뷰 항목 및 각 항목에 등록된 디멘젼 목록을 조회 할 수 있다.

-. 비즈니스 오브젝트 검색 : 비즈니스 오브젝트에서 필요한 항목을 검색하는 기능을 제공한다.

-. 피벗 행디멘젼 항목 목록 : 피벗의 행영역에 표현되는 디멘젼을 추가한다.

-. 피벗 메져 항목 목록 : 피벗 결과 그리드의 수치값에 해당하는 디멘젼/메져를 추가한다.

-. 피벗 열 항목 목록 : 피벗 결과 그리드의 헤더에 해당하는 디멘젼 항목을 추가한다.

그림의 리포트를 실행하면 다음의 그리드 결과 형태로 나타난다.

(그림 : 리포트 결과 시트 피벗 조회 결과 예제 - 그리드)
