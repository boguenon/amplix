2.3. 서버 로그 파일

서버 로그 파일

서버의 정상적인 설치 완료시 로그 파일이 logs/amplix.out.log 파일에 저장된다. 시스템 설치 이후 문제점 발생시 로그 파일의 내용을 통해 문제점 파악 후 다시 서비스를 시작한다.

로그 파일 저장에 대한 초기값은 일단위 아카이브 파일로 생성되어 logs/amplix.out.날짜.순번.log.gz 형태로 백업 저장 관리된다. 순번은 파일의 크기가 10 Mbyte 를 넘어서면 자동으로 생성되며 기존 파일은 압축된다. 최대 로그파일의 저장 기간은 30일로 설정되어 있다.

로그 설정 정보의 변경을 위해서는 다음 파일을 수정한 후 앰플릭스 서버 서비스를 재시작한다.

config/logback.xml

설정 부분 :

-. < maxFileSize> 10MB< / maxFileSize>

-. < maxHistory > 30 </ maxHistory> (디폴트값 30일, 단위 (일).)

예) logs/amplix.out.2020-01-01.0.log.gz

로그 파일의 내용은 시스템의 오류, 시스템에서 발생하는 알림 메시지를 시간 순서데로 저장된다.
